# serverless

Deploys a small Linux Function App pattern using Azure Verified Modules.

Registry modules:

- App Service Plan source: `Azure/avm-res-web-serverfarm/azurerm`
- App Service Plan version: `2.0.6`
- Function App source: `Azure/avm-res-web-site/azurerm`
- Function App version: `0.22.0`

This wrapper keeps the interface consistent with the other app modules.
